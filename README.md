# PROJETO EMPRESARIAL
Logo aqui voces podem inserir a Visao do produto, que nada mais é de que uma descrição bem objetiva do produto que esta sendo trabalhado

## PROBLEMA 
Aqui voces podem especificar o problema ou a situação que motiva a criação do produto.

## PUBLICO
Aqui voces podem colocar o publico, ou melhor, para quem o produto é direcionado.
